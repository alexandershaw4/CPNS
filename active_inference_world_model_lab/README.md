# Active Inference World Model Lab

*A visual playground for belief, uncertainty and action.*

This is a static browser demo for a simple active inference world-model agent. The V1 demo is **Find the Mug**: an agent starts with semantic priors about where mugs are usually found, evaluates possible search policies and updates its beliefs after sensory observations.

## Run

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
